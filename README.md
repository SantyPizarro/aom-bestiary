# Bestiario de Age of Mythology

Experiencia frontend de una sola pantalla para explorar las unidades miticas del
Age of Mythology original. El proyecto busca que el bestiario se sienta como una
extension de la interfaz clasica del juego y no como un catalogo web moderno.

## Experiencia

- Visor 3D interactivo con rotacion, zoom y encuadre automatico.
- Animaciones de combate y movimiento seleccionables por unidad.
- Variantes visuales para equipo, mejoras y estados especiales.
- Escenarios culturales con terreno, agua, edificios e iluminacion contextual.
- Fichas con estadisticas, descripciones, mejoras e iconografia del juego.
- Interfaz adaptable para las culturas griega, egipcia y nordica.
- Diseno responsive para escritorio y dispositivos moviles.

## Tecnologias

La aplicacion fue desarrollada con React, TypeScript, Vite, Three.js, React Three
Fiber y Drei. La escena combina modelos animados, materiales, camaras adaptativas,
controles orbitales, luces, sombras y escenarios construidos en tiempo real.

El desarrollo requirio resolver diferencias de escala y origen entre modelos,
mantener piezas de equipo sincronizadas con cada animacion, representar mejoras
visuales y conservar una interfaz densa sin quitar protagonismo al visor 3D.

## Ejecucion

```bash
npm install
npm run dev
```

Este repositorio no incluye recursos propietarios ni el proceso privado utilizado
para prepararlos. La version publica utiliza estados alternativos cuando esos
recursos no estan disponibles.
